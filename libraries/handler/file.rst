::

+-------------+------------------------+
|属性         |值                      |

:方法:


+-----------------+---------------------+
|方法名           |说明                 |
+=================+=====================+
|`__construct()`_ |构造                 |
+-----------------+---------------------+

+-----------------+---------------------+
|`close()`_       |关闭 session         |
+-----------------+---------------------+
|`read()`_        |读取 Session         |
+-----------------+---------------------+
|`write()`_       |写入 Session         |
+-----------------+---------------------+

|`gc()`_          |垃圾回收 Session     |
+-----------------+---------------------+

open()
------
打开 session

  public function open (
      string $save_path,
      string $session_name
  ) : bool


:参数:
  +-------------+----------------------+
  |名称         |说明                  |

  |save_path    |存储会话的路径        |
  +-------------+----------------------+
  |session_name |会话名称              |
